About TechDetect

TechDetect is a python opencv-powered facial detection and check in program allowing users to submit and read info from an SQL database using nothing but their faces. All data can be easily accessed here on the TechDetect website by anyone with a login, allowing monitoring of real-time data as well as previous day's attendances.
